About the implementation:
This project is an implementation of Arithmetic Units that can perform 4bit X 4bit and 2bit X 2bit multiplications.
The implementation is done in Verilog

Tool required : Xilinx ISE Design Suite

To Run this project
1. Start Xilinx ISE Design Suite 
2. Choose open project option
3. Select the folder ArithmeticUnits
4. Open the file named ArithmeticUnits.xise
5. Select Simulation under Design in top left corner
6.Select Testbench boothBrickTest.v for 2x2 multiplication or mux4x4TEST.v for 4x4 multiplication
7. Choose "Simulate Behavioral Model" under "processes" from the left middle screen
